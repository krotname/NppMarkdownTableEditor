# Markdown Table Editor logos

Production-ready logo assets for the Markdown Table Editor project.

## Selected logo

The selected direction is the document/table mark with an amber alignment guide and teal check mark:

- `selected-logo.png` - selected raster reference.
- `selected-logo.svg` - selected production SVG icon.
- `pluginIcon-40.svg` - selected compact 40x40 SVG for JetBrains `META-INF/pluginIcon.svg`.

## Files

- `markdown-table-editor-icon.svg` - main square SVG icon candidate, matching the selected logo.
- `markdown-table-editor-logo-horizontal.svg` - horizontal logo on a light background.
- `markdown-table-editor-logo-dark.svg` - horizontal logo on a dark background.

The SVG files keep the project name exact and are safer for production use than raster references with generated details.
