# Markdown Table Editor logos

Generated logo concepts and production-ready SVG candidates for the Markdown Table Editor project.

## Selected logo

The selected direction is the document/table mark with an amber alignment guide and teal check mark:

- `selected-logo.png` - selected generated raster concept.
- `selected-logo.svg` - selected production SVG icon.
- `pluginIcon-40.svg` - selected compact 40x40 SVG for JetBrains `META-INF/pluginIcon.svg`.

## Files

- `concept-grid-cursor.png` - generated raster concept, strongest small-icon direction.
- `concept-document-table.png` - generated raster concept for README or marketplace exploration.
- `concept-minimal-arrow.png` - generated raster concept with a more abstract alignment symbol.
- `markdown-table-editor-icon.svg` - main square SVG icon candidate, matching the selected logo.
- `markdown-table-editor-logo-horizontal.svg` - horizontal logo on a light background.
- `markdown-table-editor-logo-dark.svg` - horizontal logo on a dark background.

The SVG files keep the project name exact and are safer for production use than raster concepts with generated details.
