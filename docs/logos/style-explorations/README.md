# Style explorations

Alternative generated logo directions for Markdown Table Editor.

- `contact-sheet.png` - preview grid with all style variants.
- `style-minimal-monochrome.png` - strict black-and-white utility mark.
- `style-terminal.png` - terminal/editor-inspired dark mark.
- `style-windows-utility.png` - practical Windows desktop utility direction; selected as the project logo.
- `style-jetbrains-geometric.png` - bold geometric plugin-icon direction.
- `style-blueprint.png` - technical drawing / layout precision direction.
- `style-editorial-document.png` - softer document/productivity direction.
- `style-retro-pixel.png` - retro editor / pixel-art direction.

These are raster style concepts. Convert the chosen direction into SVG before using it as the production plugin icon.
